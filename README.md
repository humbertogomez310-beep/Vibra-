# Vibra 🎧🔥

Vibra es una aplicación móvil que detecta el estado de ánimo del usuario a través de su voz
y reproduce música local acorde a ese estado usando inteligencia artificial.

## Estado del proyecto
🚧 En desarrollo (fase inicial)

## Funciones principales
- Detección de estado de ánimo por voz
- Reproducción de música local
- Motor de ánimo propio (MoodEngine)
- Enfoque en experiencia emocional

## Notas
Proyecto en desarrollo desde móvil.
